WELCOME TO MY SHOP

In this project :

I used REST-Api reqests to get products from fakestoreAPI and also to post customer data in google/firebase

I devided aplication for multipiple components to make this shop easyer to work with and read

I used RxJS to make this aplication less static and to provide best expierience for the user while using the store

I used a lot of services to keep logic outside the components and make it easyer to communicate between them

I make use of Local Storage to deliver to the customer best web shoping-ish expierience

And many others smaller features that You can find during using the shop!
